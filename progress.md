# Процесс разработки конфигурации 1С
- Созданы константы НазваниеОрганизации и ТекущийУчебныйГод
- Создано перечисление СтатусыТеста
- Создано перечисление СтатусыВопроса
- Создано перечисление ФорматыЭкспорта
- Создан справочник Кафедры
- Создан справочник Факультеты (неиерархический)
- Создан справочник Дисциплины (иерархический)
- Создан справочник Темы (иерархический)
- Создан справочник УчебныеГруппы (иерархический)
- Создан справочник Сотрудники (с реквизитами)
- Создан справочник Студенты
- Создан справочник ТипыВопросов (предопределённые)
- Создан справочник УровниСложности
- Создан справочник Компетенции
- Создан справочник ШаблоныОтчётов
- Создан справочник Вопросы (основные реквизиты)
- Создан справочник ВариантыОтветов (владелец – Вопросы)
- Создан справочник Тесты
- Добавлена табличная часть ЗаданияТеста в справочник Тесты
- Создан документ РазработкаТеста
- Создан документ НазначениеТеста
- Реализовано проведение документа НазначениеТеста (регистр ОстаткиПопыток)
- Создан документ ПрохождениеТеста
- Написан модуль проведения документа ПрохождениеТеста (основная логика)
- Реализован подсчёт баллов при проведении ПрохождениеТеста
