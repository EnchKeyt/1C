- Создано перечисление СтатусыТеста
- Создано перечисление СтатусыВопроса
- Создано перечисление ФорматыЭкспорта
- Создан справочник Кафедры
- Создан справочник Факультеты (неиерархический)
- Создан справочник Дисциплины (иерархический)
- Создан справочник Темы (иерархический)
- Создан справочник УчебныеГруппы (иерархический)
- Создан справочник Сотрудники (с реквизитами)
- Создан справочник Студенты
- Создан справочник ТипыВопросов (предопределённые)
- Создан справочник УровниСложности
- Создан справочник Компетенции
- Создан справочник ШаблоныОтчётов
- Создан справочник Вопросы (основные реквизиты)
- Создан справочник ВариантыОтветов (владелец – Вопросы)
- Создан справочник Тесты
- Добавлена табличная часть ЗаданияТеста в справочник Тесты
- Создан документ РазработкаТеста
- Создан документ НазначениеТеста
- Реализовано проведение документа НазначениеТеста (регистр ОстаткиПопыток)
- Создан документ ПрохождениеТеста
- Написан модуль проведения документа ПрохождениеТеста (основная логика)
- Реализован подсчёт баллов при проведении ПрохождениеТеста
- Создан регистр накопления ОстаткиПопыток (остатки)
- Создан регистр накопления РезультатыПопыток (обороты)
- Создан регистр накопления СтатистикаВопросов (обороты)
- Создан документ ОценкаРезультата (корректировка баллов)
- Создан документ УтверждениеВопроса
- Создан бизнес-процесс СогласованиеВопроса
- Реализована отправка задач методисту в бизнес-процессе
- Создана обработка ЗаполнениеТестовыхДанных (начальная версия)
- Добавлена генерация факультетов и кафедр в обработке заполнения
- Добавлена генерация дисциплин и тем
- Добавлена генерация студентов и учебных групп
- Добавлена генерация вопросов и вариантов ответов (10+ вопросов)
- Добавлена генерация тестов и заданий (10+ тестов)
- Добавлена генерация документов ПрохождениеТеста (10+ прохождений)
- Создан отчёт СводнаяУспеваемость (СКД)
- Добавлена печатная форма .xlsx для отчёта СводнаяУспеваемость
- Создан отчёт АнализВопросов (сложный, 3+ объекта)
- Добавлена печатная форма .xlsx для отчёта АнализВопросов
- Создан отчёт РейтингПоДисциплинам (сложный)
- Добавлена печатная форма для отчёта РейтингПоДисциплинам
- Создан отчёт СписокВопросовПоТеме (простой)
- Добавлена печатная форма .docx для СписокВопросовПоТеме
- Создан отчёт ЖурналПрохождений (простой)
- Добавлена печатная форма .docx для ЖурналПрохождений
- Создана подсистема Администрирование
- Создана подсистема Разработка контента
- Создана подсистема Тестирование студентов
- Создана подсистема Отчёты и аналитика
- Создана роль Администратор (полные права)
- Создана роль Преподаватель (права на свои дисциплины)
- Создана роль Студент (ограничение по записям)
- Настроены права доступа к справочникам (чтение/запись)
